# 네 개의 화면 — p5.js 제너러티브 전시

과제 1–4의 작품을 한 페이지의 2행 2열 전시로 묶은 GitHub Pages 사이트입니다.
사이트에 접속하면 네 작품이 액자 안에서 **즉시 재생**됩니다.

## 배치 (정적 → 동적)

| | 정적 (Static) | 동적 (Motion) |
|---|---|---|
| **추상화** | 과제 1 · 기하의 균형 | 과제 4 · 숨 쉬는 기하 |
| **자화상** | 과제 2 · 자화상 | 과제 3 · 살아있는 자화상 |

- 과제 1 `abstract_art/` — 정적 추상화
- 과제 2 `caricature_portrait/` — 정적 캐리커처 자화상
- 과제 3 `assignment3/` — 인터랙티브(동적) 자화상
- 과제 4 `assignment2/` — 애니메이션(동적) 추상화

## 코멘트

같은 손에서 나온 정적·동적 작업을 나란히 걸어, 한 컷의 그림이 어떻게 시간으로 풀리는지
비교해 보고자 했습니다. 다크 갤러리 컨셉으로 작품의 골드·틸·크림슨 팔레트를
페이지 전체로 확장했습니다.

## 핵심 기술

- **HTML** — 시맨틱 `figure`/`figcaption`, `iframe` 임베드로 원작 코드를 그대로 보존
- **CSS** — Grid 2×2 레이아웃, `aspect-ratio`, 골드 라인 액자, 그레인·비네트 레이어,
  staggered fade-in 애니메이션, 반응형(모바일 1열)
- **JavaScript** — p5.js 드로잉, `ResizeObserver` 기반 캔버스 반응형 스케일링

## AI 활용

전시 페이지의 미학 방향(다크 갤러리 컨셉·타이포그래피·반응형 임베드 구조)을 다듬는 데
AI(Claude)를 협업 도구로 사용했습니다. 작품 4점의 드로잉 로직은 직접 작성했습니다.
